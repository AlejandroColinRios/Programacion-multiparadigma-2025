# Sistema de Gestion de Biblioteca

## Proposito del Programa

El Sistema para la gstion de una biblioteca es un programa desarrollado en Python que permite administrar los libros y usuarios de una biblioteca, Ofrece una interfaz de menu en consola interactiva para agregar libros y usuarios, prestar y devolver libros, listar libros disponibles y guardar el estado del sistema

## Como Ejecutar el Proyecto

Para ejecutar el proyecto, abre una terminal, entra a la carpeta `biblioteca/` y ejecuta: main.py
